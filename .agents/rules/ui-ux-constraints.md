# UI/UX & Rendering Rules
- Do not change the current styles, layout, and design of anything unless explicitly asked. This is strictly a UI/UX issue.
- Always look out for raw text strings render errors (text outside of a <Text> component) and virtualized list errors (nested scrolling lists) in the files. If found, notify me immediately.
- You can add logs using __DEV__ for debugging layouts or UI state lifecycle as well.
